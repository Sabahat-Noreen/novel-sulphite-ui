# Novel Sulphite Webpage

A sleek, lightweight landing page dedicated to the novel *Sulphite*. This project focuses on a clean presentation of the novel's themes and characters, built entirely using pure HTML and CSS.


## 📁 Project Structure

The repository is organized with a root directory containing the documentation and a dedicated source folder for the webpage assets:

```text
novel-sulphite-ui
├── README.md                 # Project documentation
└── Sulphite                  # Main project assets folder
    ├── index.html            # Webpage structure
    ├── style.css             # Custom styling and layout
    ├── um.jpg                # Character image
    └── rj.jpg                # Character image
